Brief implementation notes
- Overloading used a single method name `area` with distinct parameter lists:
  (double), (double,double), (int,int), and (int).
- Formulas: circle πr^2; rectangle w×h; triangle ½×base×height; square side^2.
- Each method validates inputs (> 0) and throws IllegalArgumentException on bad values.
- Main prints the four required lines and demonstrates an exception via try/catch.

Build/run (repro)

- Run:      java -cp out org.howard.edu.lsp.midterm.question2.Main
